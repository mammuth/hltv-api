HLTV HTTP RESTful API

Usage & Features

List upcoming matches 
/api/upcoming-matches
Get an ical / ics of upcoming matches 
/api/upcoming-matches/ical

Both endpoints can be filtered by appending query parameters like this: 

/api/upcoming-matches/?team=6665&team=7532

The ID of your wished team can be found in the URL of hltv: 

https://www.hltv.org/team/7532/big
